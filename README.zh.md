## 这个仓库

仓库里只放**可发布的静态站点**（`site/`）。纯 HTML/CSS/JS，内容打包在 `site/content.js` 里——直接用浏览器打开 `site/index.html` 就能离线阅读，无需构建、无需起服务。界面按浏览器语言自动选中英，右上角随时可切换。

部署：推送到 `main` 即把 `site/` 发布到 GitHub Pages（见 `.github/workflows/pages.yml`）；自定义域名在 `site/CNAME`。
